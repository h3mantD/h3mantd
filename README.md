### About me

I'm a full-stack software engineer with over five years of experience building everything from API packages and command-line tools to full-fledged SaaS platforms.

I like turning ideas into working code and learning new technologies along the way. On GitHub, you'll find PHP and Laravel libraries, Inertia/React applications, Vue components, CLI utilities, network/IP helpers, data-migration tools, and more.

Whether it's a back-office platform, a text transformation tool, or a package for IPv4/IPv6 operations, I enjoy solving problems with clean code and clear documentation.

---

### Focus areas

- **Web applications**: Building practical, scalable apps with Laravel, PHP, React, Vue, Inertia and TypeScript.
- **APIs & services**: REST APIs, background jobs, queues, scheduled tasks and service-oriented workflows.
- **Developer tooling**: CLI utilities, automation scripts, code helpers, package development and internal tools.
- **Data & storage**: Database design, migrations, reporting, imports, exports and query optimisation.
- **Network & security**: IP address tooling, hashing/encryption utilities, authentication flows and security-adjacent workflows.
